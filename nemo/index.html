About NEMO

NEMO is simply a SpeechRecognition interface of the Web Speech API . It is the controller interface for the recognition services and also handles the SpeechRecognitionEvent sent from the recognition service.

The Web Speech API enables you to incorporate voice data into web apps. The Web Speech API has two parts: SpeechSynthesis (Text-to-Speech), and SpeechRecognition (Asynchronous Speech Recognition.)

For further understanding visit: developer.mozilla.org
